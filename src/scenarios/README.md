# Scenarios

Her senaryo bir klasör. JSON + opsiyonel TypeScript modülü içerir.

- `insurgency/` — Rebel Inc. tarzı stabilizasyon (MVP senaryosu)
- `interstate-war/` — ülkeler arası savaş (post-MVP)
- `campaign/` — bağlantılı senaryolar zinciri (post-MVP)

Bir senaryo şunları tanımlar: başlangıç durumu, kazanma/kaybetme koşulu, düşman AI, açık/kapalı mekanikler.
